# MAP Enhancement Proposals (MEPs)
Major changes to MAP require a MAP Enhancement Proposal (MEP).

## When is an MEP required?
- Breaking protocol changes
- New SDK APIs
- Security model changes
- Governance changes

## MEP Process
1. Fork & create `rfcs/mep-XXXX-title.md`
2. Open PR with MEP
3. 14-day comment period
4. Maintainer decision (consensus)
5. Merge or close
